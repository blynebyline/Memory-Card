## Memory Card

A memory card game built with React as part of The Odin Project curriculum. The objective is to click each card only once. After every click, the cards are shuffled, making it progressively harder to remember which cards have already been selected.

# Features
  - Dynamic card generation
  - Score tracking
  - Best score tracking
  - Card shuffling after every selection
  - Game over detection for repeated selections
  - Responsive design
# Built With
  - React
  - Vite
  - JavaScript (ES6+)
  - CSS
